# Perplexity Selection Translator (English)

## Description
Firefox extension that translates selected text using Perplexity, without API tokens.

## Usage
1. Select text on any website.
2. Right-click and choose **Translate with Perplexity**.
3. A popup window opens with the translation prompt in Perplexity.
4. To change the target language, click the extension icon or open the full options page.
