# Perplexity Selection Translator (Português)

## Descrição
Extensão para Firefox que traduz texto selecionado com o Perplexity, sem tokens de API.

## Uso
1. Selecione um texto em qualquer site.
2. Clique com o botão direito e escolha **Traduzir com Perplexity**.
3. Uma janela popup será aberta com o prompt de tradução no Perplexity.
4. Para alterar o idioma de destino, clique no ícone da extensão ou abra a página completa de opções.
