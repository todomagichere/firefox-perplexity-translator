# Perplexity Selection Translator (Français)

## Description
Extension Firefox qui traduit le texte sélectionné avec Perplexity, sans jetons API.

## Utilisation
1. Sélectionnez du texte sur un site web.
2. Faites un clic droit et choisissez **Traduire avec Perplexity**.
3. Une fenêtre popup s'ouvre avec l'invite de traduction dans Perplexity.
4. Pour changer la langue cible, cliquez sur l'icône de l'extension ou ouvrez la page complète des options.
