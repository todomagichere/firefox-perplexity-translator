# Perplexity Selection Translator (Español)

## Descripción
Extensión de Firefox que traduce el texto seleccionado usando Perplexity, sin usar tokens de API.

## Uso
1. Selecciona texto en cualquier sitio web.
2. Haz clic derecho y elige **Traducir con Perplexity**.
3. Se abrirá una ventana emergente con el prompt de traducción en Perplexity.
4. Para cambiar el idioma de destino, haz clic en el icono de la extensión o abre la página completa de opciones.
