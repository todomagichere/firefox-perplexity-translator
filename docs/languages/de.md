# Perplexity Selection Translator (Deutsch)

## Beschreibung
Firefox-Erweiterung zum Übersetzen markierten Textes mit Perplexity, ohne API-Token.

## Verwendung
1. Markiere Text auf einer Webseite.
2. Klicke mit der rechten Maustaste und wähle **Mit Perplexity übersetzen**.
3. Ein Popup-Fenster mit dem Übersetzungs-Prompt in Perplexity wird geöffnet.
4. Um die Zielsprache zu ändern, klicke auf das Erweiterungs-Icon oder öffne die vollständige Optionsseite.
