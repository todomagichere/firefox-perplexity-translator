# Perplexity Selection Translator (Italiano)

## Descrizione
Estensione Firefox che traduce il testo selezionato con Perplexity, senza token API.

## Utilizzo
1. Seleziona del testo su qualsiasi sito web.
2. Fai clic destro e scegli **Traduci con Perplexity**.
3. Si apre una finestra popup con il prompt di traduzione su Perplexity.
4. Per cambiare la lingua di destinazione, fai clic sull'icona dell'estensione o apri la pagina opzioni completa.
